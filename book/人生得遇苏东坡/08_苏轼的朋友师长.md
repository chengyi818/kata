## 👨‍🏫 师长伯乐

### 欧阳修：知遇之恩，没齿难忘

| 信息 | 详情 |
|:---|:---|
| 姓名 | 欧阳修，字永叔，号醉翁、六一居士 |
| 生卒 | 1007年 - 1072年 |
| 身份 | 北宋文坛领袖，唐宋八大家之一 |
| 与苏轼关系 | 恩师、伯乐 |

#### 科举识才

1057年，欧阳修主持科举考试。当他读到苏轼的文章时，惊为天人：

> **"读轼书，不觉汗出，快哉快哉！老夫当避路，放他出一头地也。"**

欧阳修本想将苏轼定为第一，但怀疑这篇文章是自己学生曾巩所写，为避嫌将其列为第二。后来得知是苏轼所作，更加赏识。

#### 预言成真

欧阳修曾对人说：

> **"此人他日文章，必独步天下。"**

又说：

> **"三十年后，世人只知苏轼，不知有我欧阳修也。"**

果然，苏轼后来成为一代文宗，欧阳修的预言完全应验。

#### 传承衣钵

欧阳修不仅赏识苏轼的才华，更在仕途上大力提携。可以说，没有欧阳修，就没有后来的苏东坡。

## 📚 苏门四学士

苏轼在文坛的地位确立后，许多年轻才俊拜入门下，其中最出名的是"苏门四学士"。

### 黄庭坚：并称"苏黄"

| 信息 | 详情 |
|:---|:---|
| 姓名 | 黄庭坚，字鲁直，号山谷道人 |
| 生卒 | 1045年 - 1105年 |
| 身份 | 北宋著名诗人、书法家 |
| 与苏轼关系 | 门生，诗坛双璧 |

#### 诗坛双璧

黄庭坚的诗歌成就极高，与苏轼并称**"苏黄"**，是宋诗的代表人物。

#### 书法大家

黄庭坚的书法与苏轼、米芾、蔡襄并称**"宋四家"**。

#### 师生情深

黄庭坚对苏轼极为敬重，曾说：

> **"东坡文章妙天下，忠义贯日月。"**

苏轼被贬时，黄庭坚也受到牵连，但从不后悔拜苏轼为师。

## 🙏 方外之交

### 佛印：禅机妙趣

| 信息 | 详情 |
|:---|:---|
| 姓名 | 佛印，法名了元 |
| 生卒 | 1032年 - 1098年 |
| 身份 | 禅宗高僧，金山寺住持 |
| 与苏轼关系 | 方外挚友 |

苏轼与佛印的交往，留下了许多妙趣横生的故事。

#### 典故一：八风吹不动

苏轼曾写诗自夸修行境界：

> "稽首天中天，毫光照大千。  
> 八风吹不动，端坐紫金莲。"

派人送给佛印看。佛印看后，在诗上批了两个字：**"放屁"**。

苏轼大怒，亲自过江找佛印理论。佛印笑道：

> **"八风吹不动，一屁过江来。"**

苏轼这才明白自己的修行还差得远。

#### 典故二：狗啃河上骨

一日，苏轼与佛印泛舟江上，看见一条狗在啃骨头。苏轼想取笑佛印，吟道：

> "狗啃河上（和尚）骨。"

佛印不动声色，把苏轼写的诗稿扔进水里，回应道：

> "水流东坡诗（尸）。"

两人相视大笑。

#### 典故三：你看我像什么

苏轼问佛印："你看我像什么？"

佛印说："我看你像一尊佛。"

苏轼大笑，说："我看你像一坨屎。"

回家后，苏小妹问他为何这么高兴。苏轼得意地讲了此事。

苏小妹说：

> **"佛印心中有佛，所以看你像佛。你心中有屎，所以看他像屎。"**

苏轼哑然。

## 🌙 患难之交

### 马梦得（马正卿）：黄州知己

| 信息 | 详情 |
|:---|:---|
| 姓名 | 马正卿，字梦得 |
| 身份 | 黄州旧识 |
| 与苏轼关系 | 患难知己 |

#### 助垦东坡

苏轼被贬黄州时，没有薪俸，生活困顿。马梦得四处奔走，帮苏轼申请到了城东的一片荒地。

苏轼亲自开垦这片土地，种植庄稼，并以此地自号**"东坡居士"**。

可以说，没有马梦得，就没有"东坡"这个名号。

#### 深情厚谊

苏轼曾感慨地说：

> **"马梦得与仆同年友，同年又同日生，少时多难，晚乃安佚。"**

---

### 张怀民：夜游知己

| 信息 | 详情 |
|:---|:---|
| 姓名 | 张怀民，字梦得 |
| 身份 | 同被贬谪的官员 |
| 与苏轼关系 | 黄州同命人 |

#### 《记承天寺夜游》

张怀民是苏轼在黄州最重要的朋友之一。两人都是被贬谪之人，同病相怜。

苏轼写下的《记承天寺夜游》，记录了他们的友情：

> 元丰六年十月十二日夜，解衣欲睡，月色入户，欣然起行。念无与为乐者，遂至承天寺寻张怀民。怀民亦未寝，相与步于中庭。
>
> 庭下如积水空明，水中藻、荇交横，盖竹柏影也。
>
> **何夜无月？何处无竹柏？但少闲人如吾两人者耳。**

"闲人"二字，道尽了两个失意文人的自嘲与豁达。

---

### 陈季常：河东狮吼

| 信息 | 详情 |
|:---|:---|
| 姓名 | 陈慥，字季常 |
| 身份 | 隐士，苏轼旧友 |
| 与苏轼关系 | 黄州邻居 |

#### 河东狮吼

陈季常是苏轼在黄州的邻居和好友。他的妻子柳氏性格彪悍，每当陈季常请客饮酒，柳氏就在隔壁大声喊叫。

苏轼为此写诗调侃：

> "龙丘居士亦可怜，谈空说有夜不眠。  
> 忽闻河东狮子吼，拄杖落手心茫然。"

**"河东狮吼"**这个成语，就是从这里来的，用来形容妻子凶悍。

## ⚔️ 亦敌亦友

### 王安石：政敌与文友

| 信息 | 详情 |
|:---|:---|
| 姓名 | 王安石，字介甫，号半山 |
| 生卒 | 1021年 - 1086年 |
| 身份 | 北宋宰相，唐宋八大家之一 |
| 与苏轼关系 | 政见不合，文心相惜 |

#### 政见不合

王安石主持变法，苏轼反对新法中的弊端，两人在政治上是对立的。苏轼因此受到新党打压，仕途坎坷。

#### 文心相惜

但在文学上，两人互相欣赏。王安石曾赞苏轼：

> **"不知更几百年，方有如此人物。"**

#### 晚年和解

王安石罢相后隐居金陵，苏轼路过时特意拜访。两人抛开政见，畅谈文学，相见甚欢。

王安石感叹：

> **"不知更几百年，方有如此人物！"**

这段"相逢一笑泯恩仇"的故事，成为千古佳话。
