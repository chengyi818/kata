## 👴 祖父：苏序

| 信息 | 详情 |
|:---|:---|
| 姓名 | 苏序 |
| 籍贯 | 四川眉山 |
| 性格 | 豪爽大方，乐善好施 |

苏序是苏轼的祖父，虽然史料记载不多，但他对苏氏家族的文化传承有重要影响。据载，苏序性格豪爽，喜欢结交朋友，在乡里颇有声望。

苏轼曾在文章中提到祖父：
> "吾祖父游宦四方，性度豁达，轻财好施。"

## 👨 父亲：苏洵（老苏）

| 信息 | 详情 |
|:---|:---|
| 姓名 | 苏洵，字明允，号老泉 |
| 生卒 | 1009年 - 1066年（享年57岁） |
| 籍贯 | 四川眉山 |
| 身份 | 北宋文学家，唐宋八大家之一 |
| 代表作 | 《嘉祐集》《六国论》《权书》 |

### 大器晚成

苏洵27岁前不爱读书，游荡乡里。27岁时幡然醒悟，开始发愤读书，最终成为一代文学大家。

> "苏老泉，二十七，始发愤，读书籍。" ——《三字经》

### 携子进京

1056年（嘉祐元年），苏洵携苏轼、苏辙二子进京应试，受到欧阳修的大力推荐，父子三人名动京城。

### 三苏并称

苏洵与苏轼、苏辙并称**"三苏"**，同列**唐宋八大家**，这在文学史上是绝无仅有的。

```

│  唐代：韩愈、柳宗元                                     │
│  宋代：欧阳修、王安石、曾巩、苏洵、苏轼、苏辙          │
```

## 👩 母亲：程夫人

| 信息 | 详情 |
|:---|:---|
| 姓名 | 程氏（名不详） |
| 出身 | 眉山大理寺丞程文应之女 |
| 特点 | 知书达理，教子有方 |

### 书香门第

程夫人出身于书香世家，自幼饱读诗书，嫁入苏家后，成为家中的精神支柱。

### 教子有方

程夫人非常重视对苏轼、苏辙兄弟的教育。她亲自教导二子读书识字，培养他们的品德和学识。

**著名典故：读《范滂传》**

程夫人曾给苏轼讲述东汉名臣范滂的故事。范滂为官清正，不惧权贵，最终为正义献身。

苏轼听后问母亲：
> "母亲，我如果将来像范滂一样，您会怎么样？"

程夫人回答：
> "你能做范滂，我难道不能做范滂的母亲吗？"

这个故事深深影响了苏轼一生的为人处世。

## 👦 弟弟：苏辙（小苏）

| 信息 | 详情 |
|:---|:---|
| 姓名 | 苏辙，字子由，号颍滨遗老 |
| 生卒 | 1039年 - 1112年（享年73岁） |
| 身份 | 北宋文学家、政治家，唐宋八大家之一 |
| 最高官职 | 尚书右丞（副宰相级别） |
| 代表作 | 《栾城集》 |

### 兄弟情深

苏轼与苏辙兄弟情深，堪称古代兄弟情谊的典范：

- 📝 二人诗文唱和，留下大量书信往来
- 🤝 乌台诗案时，苏辙愿削官救兄
- 🌙 苏轼写《水调歌头》时，正思念弟弟
- ⚰️ 苏轼遗愿葬于汝州，只为离弟弟近些

### 性格对比

| 方面 | 苏轼 | 苏辙 |
|:---:|:---:|:---:|
| 性格 | 豪放不羁 | 沉稳内敛 |
| 文风 | 汪洋恣肆 | 淡雅朴素 |
| 仕途 | 起伏跌宕 | 相对平稳 |
| 处世 | 锋芒毕露 | 谨慎圆融 |

苏辙曾评价兄长：
> "东坡何罪？独以名太高。"

## 💕 三位伴侣

### 第一任妻子：王弗

| 信息 | 详情 |
|:---|:---|
| 姓名 | 王弗 |
| 生卒 | 1039年 - 1065年（享年27岁） |
| 籍贯 | 四川眉州青神 |
| 父亲 | 乡贡进士王方 |
| 婚龄 | 11年（1054-1065） |
| 子嗣 | 苏迈 |

**才貌双全，知书达理**

王弗自幼随父读书，与苏轼结婚时年仅16岁。她聪慧贤惠，是苏轼的贤内助。

**红袖添香，幕后听言**

王弗常在苏轼读书时相伴左右。每当苏轼有访客，她便在屏风后静听，事后为苏轼分析来客的品性和用意，见解往往精准。

**英年早逝**

1065年，王弗病逝于京城，年仅27岁。苏轼悲痛欲绝。

**十年悼亡**

1075年正月，苏轼在密州梦见亡妻，写下千古悼亡词：

> **十年生死两茫茫，不思量，自难忘。**  
> **千里孤坟，无处话凄凉。**  
> **纵使相逢应不识，尘满面，鬓如霜。**
>
> **夜来幽梦忽还乡，小轩窗，正梳妆。**  
> **相顾无言，惟有泪千行。**  
> **料得年年肠断处，明月夜，短松冈。**

### 第二任妻子：王闰之

| 信息 | 详情 |
|:---|:---|
| 姓名 | 王闰之 |
| 生卒 | 1048年 - 1093年（享年46岁） |
| 与王弗关系 | 堂妹（一说堂姐） |
| 婚龄 | 25年（1068-1093） |
| 子嗣 | 苏迨、苏过 |

**续弦再娶**

王弗去世三年后（1068年），苏轼续娶了王弗的堂妹王闰之，时年王闰之21岁，苏轼32岁。

**贤惠持家**

王闰之性格温柔贤惠，不如王弗聪慧，但持家有方。她陪伴苏轼度过了仕途最艰难的岁月：

- 🌊 乌台诗案的惊恐
- 🏚️ 黄州贬谪的困苦
- 📈 回京任职的短暂辉煌
- 📉 再次外放的漂泊

**先于苏轼离世**

1093年，王闰之病逝于京城，苏轼亲自撰写祭文悼念。

---

### 侍妾：王朝云

| 信息 | 详情 |
|:---|:---|
| 姓名 | 王朝云，字子霞 |
| 生卒 | 1062年 - 1096年（享年34岁） |
| 出身 | 杭州歌妓 |
| 入苏门时间 | 1074年（苏轼任杭州通判时） |
| 子嗣 | 苏遁（幼年夭折） |

**西湖相遇**

1074年，苏轼在杭州任通判时，于西湖边遇见了年仅12岁的歌女王朝云，将其买入府中，初为侍女。

**红颜知己**

朝云聪慧灵秀，精通诗词歌赋，是苏轼真正的红颜知己。苏轼曾问身边人："我肚子里装的是什么？"

- 有人说是"文章"
- 有人说是"见识"

唯有朝云回答：
> **"学士一肚皮不合时宜。"**

苏轼大笑称是。

**患难与共**

朝云是唯一陪伴苏轼走完贬谪之路的女性：

```
1080年 ── 随苏轼贬谪黄州
1094年 ── 随苏轼贬谪惠州
1096年 ── 病逝于惠州（苏轼未赴儋州前）
```

**香消玉殒**

1096年，朝云因瘟疫病逝于惠州，年仅34岁。苏轼将她葬于惠州西湖边的栖禅寺旁，亲题墓志铭：

> **"不合时宜，惟有朝云能识我；  
> 独弹古调，每逢暮雨倍思卿。"**

## 👶 苏轼的儿子们

| 儿子 | 母亲 | 简介 |
|:---|:---|:---|
| **苏迈** | 王弗 | 长子，曾任酸枣县尉 |
| **苏迨** | 王闰之 | 次子，曾任承务郎 |
| **苏过** | 王闰之 | 三子，最有才华，有"小坡"之称 |

三子苏过最得苏轼真传，文学造诣最高，被称为"小坡"。他陪伴父亲度过了最后的贬谪岁月，在海南儋州侍奉左右。
