## 📜 经典诗词全文

### 《水调歌头·明月几时有》

**背景**：1076年中秋，苏轼在密州任知州，与弟弟苏辙分别七年，对月怀人。

```
丙辰中秋，欢饮达旦，大醉，作此篇，兼怀子由。

明月几时有？把酒问青天。
不知天上宫阙，今夕是何年。
我欲乘风归去，又恐琼楼玉宇，高处不胜寒。
起舞弄清影，何似在人间。

转朱阁，低绮户，照无眠。
不应有恨，何事长向别时圆？
人有悲欢离合，月有阴晴圆缺，此事古难全。
但愿人长久，千里共婵娟。
```

---

### 《念奴娇·赤壁怀古》

**背景**：1082年，苏轼被贬黄州期间，游览赤壁古战场，怀古伤今。

```
大江东去，浪淘尽，千古风流人物。
故垒西边，人道是，三国周郎赤壁。
乱石穿空，惊涛拍岸，卷起千堆雪。
江山如画，一时多少豪杰。

遥想公瑾当年，小乔初嫁了，雄姿英发。
羽扇纶巾，谈笑间，樯橹灰飞烟灭。
故国神游，多情应笑我，早生华发。
人生如梦，一尊还酹江月。
```

---

### 《定风波·莫听穿林打叶声》

**背景**：1082年春，苏轼在黄州沙湖道中遇雨，有感而作。

```
三月七日，沙湖道中遇雨。雨具先去，同行皆狼狈，余独不觉，
已而遂晴，故作此词。

莫听穿林打叶声，何妨吟啸且徐行。
竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生。

料峭春风吹酒醒，微冷，山头斜照却相迎。
回首向来萧瑟处，归去，也无风雨也无晴。
```

---

### 《江城子·密州出猎》

**背景**：1075年，苏轼在密州任知州，冬日出猎，豪情满怀。

```
老夫聊发少年狂，左牵黄，右擎苍。
锦帽貂裘，千骑卷平冈。
为报倾城随太守，亲射虎，看孙郎。

酒酣胸胆尚开张，鬓微霜，又何妨！
持节云中，何日遣冯唐？
会挽雕弓如满月，西北望，射天狼。
```

---

### 《江城子·乙卯正月二十日夜记梦》

**背景**：1075年正月，苏轼梦见亡妻王弗，写下这首悼亡词。

```
十年生死两茫茫，不思量，自难忘。
千里孤坟，无处话凄凉。
纵使相逢应不识，尘满面，鬓如霜。

夜来幽梦忽还乡，小轩窗，正梳妆。
相顾无言，惟有泪千行。
料得年年肠断处，明月夜，短松冈。
```

---

### 《临江仙·夜饮东坡醒复醉》

**背景**：1082年，苏轼在黄州东坡雪堂饮酒，夜归有感。

```
夜饮东坡醒复醉，归来仿佛三更。
家童鼻息已雷鸣。
敲门都不应，倚杖听江声。

长恨此身非我有，何时忘却营营？
夜阑风静縠纹平。
小舟从此逝，江海寄余生。
```

## 💎 金句集锦

### 关于人生态度

| 金句 | 出处 | 解读 |
|:---|:---|:---|
| 竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生。 | 《定风波》 | 从容面对风雨，无畏人生起伏 |
| 也无风雨也无晴。 | 《定风波》 | 超越得失，心境平和 |
| 人生如梦，一尊还酹江月。 | 《念奴娇》 | 看淡功名，活在当下 |
| 人间有味是清欢。 | 《浣溪沙》 | 简单生活中的真正快乐 |

### 关于思念与离别

| 金句 | 出处 | 解读 |
|:---|:---|:---|
| 但愿人长久，千里共婵娟。 | 《水调歌头》 | 距离阻隔不了心灵的相通 |
| 十年生死两茫茫，不思量，自难忘。 | 《江城子》 | 深情难忘，刻骨铭心 |
| 此心安处是吾乡。 | 《定风波》 | 心安即是归处 |

### 关于豁达与超脱

| 金句 | 出处 | 解读 |
|:---|:---|:---|
| 问汝平生功业，黄州惠州儋州。 | 《自题金山画像》 | 苦难成就人生，困境孕育伟大 |
| 日啖荔枝三百颗，不辞长作岭南人。 | 《惠州一绝》 | 乐观面对贬谪，随遇而安 |
| 九死南荒吾不恨，兹游奇绝冠平生。 | 《六月二十日夜渡海》 | 把流放当作旅行，化苦难为风景 |

### 关于自然与审美

| 金句 | 出处 | 解读 |
|:---|:---|:---|
| 欲把西湖比西子，淡妆浓抹总相宜。 | 《饮湖上初晴后雨》 | 自然之美，浑然天成 |
| 不识庐山真面目，只缘身在此山中。 | 《题西林壁》 | 跳出局限，才能看清全貌 |
| 横看成岭侧成峰，远近高低各不同。 | 《题西林壁》 | 角度不同，所见不同 |

## 📖 散文名篇节选

### 《前赤壁赋》节选

> 壬戌之秋，七月既望，苏子与客泛舟游于赤壁之下。清风徐来，水波不兴。举酒属客，诵明月之诗，歌窈窕之章。少焉，月出于东山之上，徘徊于斗牛之间。白露横江，水光接天。纵一苇之所如，凌万顷之茫然。浩浩乎如冯虚御风，而不知其所止；飘飘乎如遗世独立，羽化而登仙。

> ......

> 客亦知夫水与月乎？逝者如斯，而未尝往也；盈虚者如彼，而卒莫消长也。盖将自其变者而观之，则天地曾不能以一瞬；自其不变者而观之，则物与我皆无尽也，而又何羡乎！

### 《记承天寺夜游》

> 元丰六年十月十二日夜，解衣欲睡，月色入户，欣然起行。念无与为乐者，遂至承天寺寻张怀民。怀民亦未寝，相与步于中庭。
>
> 庭下如积水空明，水中藻、荇交横，盖竹柏影也。
>
> **何夜无月？何处无竹柏？但少闲人如吾两人者耳。**

## 🎯 我的收藏

在阅读过程中，最打动我的几句话：

1. **"一蓑烟雨任平生"** —— 面对人生风雨的最佳态度

2. **"此心安处是吾乡"** —— 心安之处，便是归宿

3. **"人间有味是清欢"** —— 简单生活，才是真正的快乐

4. **"也无风雨也无晴"** —— 超越得失的最高境界

5. **"问汝平生功业，黄州惠州儋州"** —— 困境成就人生
